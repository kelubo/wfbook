## 学习路径

### 入门推荐

1. **物联网入门**：从 [传感器.tex](file:///d:/Git/wfbook/物联网/传感器/传感器.tex) 开始，了解传感器基础，然后学习 [MQTT.md](file:///d:/Git/wfbook/物联网/MQTT.md) 和 [实战项目案例.md](file:///d:/Git/wfbook/物联网/实战项目案例.md)

2. **AI入门**：从 [数据科学.md](file:///d:/Git/wfbook/AI/数据科学.md) 开始，学习数据处理，然后学习 [深度学习框架.md](file:///d:/Git/wfbook/AI/深度学习框架.md)

3. **网络入门**：从 [网络监控.md](file:///d:/Git/wfbook/网络/网络监控.md) 开始，了解网络监控工具，然后学习 [BGP.md](file:///d:/Git/wfbook/网络/BGP.md)

### 进阶推荐

1. **物联网进阶**：学习 [边缘计算.md](file:///d:/Git/wfbook/物联网/边缘计算.md)、[工业物联网.md](file:///d:/Git/wfbook/物联网/工业物联网.md)、[数字孪生.md](file:///d:/Git/wfbook/物联网/数字孪生.md)

2. **AI进阶**：学习 [边缘AI.md](file:///d:/Git/wfbook/AI/边缘AI.md)、[向量数据库.md](file:///d:/Git/wfbook/AI/向量数据库.md)

3. **架构进阶**：学习 [设计模式与架构.md](file:///d:/Git/wfbook/设计模式/设计模式与架构.md)、[云原生架构.md](file:///d:/Git/wfbook/云计算/云原生架构.md)
