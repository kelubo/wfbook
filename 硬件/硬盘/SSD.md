## 颗粒

| 颗粒类型 | 擦写次数（P/E） | 性能特征                                        | 主要应用                  |
| -------- | --------------- | ----------------------------------------------- | ------------------------- |
| SLC      | 100000          | 速度最快、性能最好、寿命最长、造价最贵。        | 一般用于企业级使用场景。  |
| MLC      | 3000-5000       | 速度、质量、寿命都稍次于SLC，但成本确不低。     | 多用于工业存储中。        |
| TLC      | 500-2000        | 速度、质量、寿命都不及SLC/MLC，但成本要低得多。 | 主流消费类 SSD 中最常见。 |
| QLC      | 100-300         | 使用寿命更短，容量相对更大，成本更低。          | 主要用于低端 SSD 中。     |
| PLC      | ＜300           | 该颗粒技术还不成熟，容量会更大，成本也更低。    | 将进一步取代 HDD 。       |

## 颗粒类型选择建议

常见固态硬盘颗粒类型排行：**SLC > MLC > TLC > QLC**

- **SLC、MLC**：价格昂贵，仅适合企业级应用
- **TLC**：价格适中，是性价比之选，推荐作为主流选择
- **QLC**：读取尚可，但写入速度慢、寿命短，不推荐用于频繁写入场景

## 常见型号避坑指南

### 不推荐型号

| 品牌型号 | 避坑原因 |
|----------|----------|
| 金士顿 NV2 | 老生常谈的网红盘，奸商专用固态硬盘 |
| 宏碁 N7000 | 小白最容易买错，注意 N7000 不是 GM7000 |
| 威刚 翼龙 S70Q | QLC 颗粒，寿命短 |
| 雷克沙 NM610Pro | QLC 颗粒，寿命短，但价格便宜 |
| 英睿达 P3 Plus | 颗粒混发，购买前一定要问清楚 |
| 海力士 P41 Plus | QLC 颗粒，注意与 P41（TLC）区分 |
| 佰维 NV7200 | QLC 颗粒，与致态 Ti600 同款方案 |
| 致态 Ti600 | QLC 颗粒 |

### 购买建议

1. **优先选择 TLC 颗粒**：兼顾性能和寿命，适合大多数用户
2. **避免 QLC 颗粒**：除非仅用于冷数据存储且预算有限
3. **注意型号区分**：有些型号名称相似但颗粒类型不同（如 P41 vs P41 Plus）
4. **确认颗粒信息**：购买前询问商家或查看官方规格，避免颗粒混发
