# Gamepad 遥控小车

[TOC]

## 概述

基于 Arduino Uno R3 的游戏手柄遥控小车项目。

## 硬件清单

| 部件 | 数量 | 说明 |
|------|------|------|
| Arduino Uno R3 | 1 | 主控板 |
| L298N 电机驱动模块 | 1 | 双路电机驱动 |
| PS2 游戏手柄 | 1 | 无线手柄 |
| PS2 接收器模块 | 1 | 2.4G 接收器 |
| 直流减速电机 | 2 | 左右轮驱动 |
| 车轮 | 3 | 2 个驱动轮 + 1 个万向轮 |
| 12V 电池组 | 1 | 供电电源 |
| 杜邦线 | 若干 | 连接导线 |

## 引脚连接

### L298N 电机驱动模块

| L298N 引脚 | Arduino 引脚 | 功能 |
|------------|-------------|------|
| ENA | D9 | 电机A PWM速度控制 |
| IN1 | D8 | 电机A方向控制1 |
| IN2 | D7 | 电机A方向控制2 |
| ENB | D10 | 电机B PWM速度控制 |
| IN3 | D6 | 电机B方向控制1 |
| IN4 | D5 | 电机B方向控制2 |
| GND | GND | 地线 |
| 12V | 电池正极 | 电机供电 |
| 5V | 5V | 可选，给Arduino供电 |

### PS2 接收器模块

| PS2 引脚 | Arduino 引脚 | 功能 |
|----------|-------------|------|
| DO | D12 | 数据输出 |
| DI | D11 | 数据输入 |
| CS | D10 | 片选信号 |
| CLK | D13 | 时钟信号 |
| GND | GND | 地线 |
| VCC | 3.3V | 电源 |

## 软件安装

### 安装 PS2X 库

1. 打开 Arduino IDE
2. 点击「项目」→「加载库」→「管理库」
3. 搜索 `PS2X_lib`
4. 安装库

### 上传代码

1. 连接 Arduino Uno 到电脑
2. 打开 `Gamepad_RC_Car.ino` 文件
3. 选择正确的端口和板型
4. 点击上传按钮

## 使用方法

### 手柄操作

| 操作 | 功能 |
|------|------|
| 左摇杆 ↑ | 前进 |
| 左摇杆 ↓ | 后退 |
| 右摇杆 ← | 左转 |
| 右摇杆 → | 右转 |
| L2 按键 | 降低速度档位 |
| R2 按键 | 提高速度档位 |
| 圆形按钮 | 紧急停止 |

### 速度档位

- 0档：停止
- 1档：约25%速度
- 2档：约50%速度
- 3档：约75%速度（默认）
- 4档：约100%速度

## 串口调试

打开串口监视器（波特率9600），可以看到：
- 手柄连接状态
- 当前速度档位

## 注意事项

1. **电源安全**：确保电池电压为12V，不要超过L298N的额定电压
2. **电机接线**：注意电机正负极，接反会导致方向相反
3. **手柄配对**：首次使用需要将手柄与接收器配对
4. **散热**：长时间高速运行时L298N会发热，建议加装散热片
5. **电池保护**：不要过度放电，建议使用带保护板的电池

## 故障排除

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 手柄无响应 | 未配对或电池没电 | 重新配对手柄，更换电池 |
| 电机不转 | 接线错误或电源不足 | 检查接线，确保电源正常 |
| 电机抖动 | PWM频率问题 | 检查引脚是否支持PWM |
| 转向异常 | 电机接线错误 | 交换电机接线 |
