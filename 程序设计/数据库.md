# 数据库

## 概述

数据库是存储和管理数据的系统，是现代应用程序的核心组件。

## 数据库分类

### 关系型数据库

- **特点**：使用表格结构，支持SQL查询，ACID事务
- **代表**：MySQL、PostgreSQL、SQLite、Oracle

### 非关系型数据库

- **特点**：灵活的数据模型，高可扩展性
- **代表**：MongoDB、Redis、Cassandra、Elasticsearch

### 时序数据库

- **特点**：专门存储时间序列数据
- **代表**：InfluxDB、TimescaleDB、Prometheus

### 图数据库

- **特点**：存储和查询图结构数据
- **代表**：Neo4j、ArangoDB

## MySQL

### 安装

```bash
# Linux
sudo apt-get install mysql-server

# macOS
brew install mysql

# 启动服务
sudo systemctl start mysql
```

### 基本操作

```sql
-- 创建数据库
CREATE DATABASE mydb;

-- 使用数据库
USE mydb;

-- 创建表
CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(100) UNIQUE NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 插入数据
INSERT INTO users (name, email) VALUES ('John', 'john@example.com');

-- 查询数据
SELECT * FROM users;

-- 更新数据
UPDATE users SET name = 'John Doe' WHERE id = 1;

-- 删除数据
DELETE FROM users WHERE id = 1;

-- 创建索引
CREATE INDEX idx_email ON users(email);
```

### Python连接MySQL

```python
import mysql.connector

connection = mysql.connector.connect(
    host='localhost',
    user='root',
    password='password',
    database='mydb'
)

cursor = connection.cursor()
cursor.execute('SELECT * FROM users')
users = cursor.fetchall()

for user in users:
    print(user)

connection.close()
```

-- 创建数据库
CREATE DATABASE mydb;

    id SERIAL PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(100) UNIQUE NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

INSERT INTO users (name, email) VALUES ('John', 'john@example.com');

SELECT * FROM users;

CREATE VIEW active_users AS
SELECT * FROM users WHERE active = true;

CREATE FUNCTION get_user_count() RETURNS INTEGER AS $$
BEGIN

END;

## SQLite

### 特点

- 轻量级，无需独立服务
- 单个文件存储
- 适合嵌入式应用和小型项目

### 基本操作

```python
import sqlite3

# 连接数据库（不存在则创建）
conn = sqlite3.connect('example.db')
cursor = conn.cursor()

# 创建表
cursor.execute('''
    CREATE TABLE IF NOT EXISTS users (
        id INTEGER PRIMARY KEY AUTOINCREMENT,
        name TEXT NOT NULL,
        email TEXT UNIQUE NOT NULL
    )
''')

# 插入数据
cursor.execute('INSERT INTO users (name, email) VALUES (?, ?)', ('John', 'john@example.com'))
conn.commit()

# 查询数据
cursor.execute('SELECT * FROM users')
users = cursor.fetchall()

for user in users:
    print(user)

conn.close()
```

## MongoDB

### 安装

```bash
# Linux
sudo apt-get install mongodb

# macOS
brew install mongodb-community

# 启动服务
sudo systemctl start mongod
```

### 基本操作

```python
from pymongo import MongoClient

# 连接MongoDB
client = MongoClient('mongodb://localhost:27017/')

# 选择数据库
db = client['mydb']

# 选择集合
users = db['users']

# 插入文档
user = {
    'name': 'John Doe',
    'email': 'john@example.com',
    'age': 30,
    'hobbies': ['reading', 'hiking']
}
users.insert_one(user)

# 查询文档
result = users.find_one({'name': 'John Doe'})
print(result)

# 更新文档
users.update_one({'name': 'John Doe'}, {'$set': {'age': 31}})

# 删除文档
users.delete_one({'name': 'John Doe'})

# 创建索引
users.create_index('email', unique=True)
```

### 基本操作

```python
import redis

# 连接Redis
r = redis.Redis(host='localhost', port=6379, db=0)

# 字符串操作
r.set('name', 'John Doe')
print(r.get('name'))

# 哈希操作
r.hset('user:1', 'name', 'John')
r.hset('user:1', 'email', 'john@example.com')
print(r.hgetall('user:1'))

# 列表操作
r.lpush('tasks', 'task1')
r.lpush('tasks', 'task2')
print(r.lrange('tasks', 0, -1))

# 集合操作
r.sadd('tags', 'python')
r.sadd('tags', 'redis')
print(r.smembers('tags'))

# 过期设置
r.set('temp_data', 'value', ex=60)
```

## 数据库设计

### 范式

- **第一范式（1NF）**：列不可再分
- **第二范式（2NF）**：非主键列完全依赖主键
- **第三范式（3NF）**：非主键列不传递依赖主键

### ER图

```
用户（User）
├── id（主键）
├── name
├── email
└── created_at

订单（Order）
├── id（主键）
├── user_id（外键→User.id）
├── total_amount
└── created_at

订单详情（OrderItem）
├── id（主键）
├── order_id（外键→Order.id）
├── product_id（外键→Product.id）
├── quantity
└── price
```

### 索引优化

```sql
-- 复合索引
CREATE INDEX idx_user_email_name ON users(email, name);

-- 覆盖索引
CREATE INDEX idx_order_user_total ON orders(user_id) INCLUDE (total_amount);

-- 唯一索引
CREATE UNIQUE INDEX idx_user_email ON users(email);
```

## 数据库事务

### ACID特性

- **原子性（Atomicity）**：事务要么全部完成，要么全部回滚
- **一致性（Consistency）**：事务前后数据完整性保持一致
- **隔离性（Isolation）**：事务之间相互隔离
- **持久性（Durability）**：事务提交后数据永久保存

### 事务示例

```python
import psycopg2

connection = psycopg2.connect(
    host='localhost',
    user='postgres',
    password='password',
    database='mydb'
)

try:
    cursor = connection.cursor()
    
    # 开始事务
    connection.autocommit = False
    
    # 执行操作
    cursor.execute('UPDATE accounts SET balance = balance - 100 WHERE id = 1')
    cursor.execute('UPDATE accounts SET balance = balance + 100 WHERE id = 2')
    
    # 提交事务
    connection.commit()
except Exception as e:
    # 回滚事务
    connection.rollback()
    print(f"Error: {e}")
finally:
    connection.close()
```

## 数据库备份与恢复

### MySQL备份

```bash
# 备份数据库
mysqldump -u root -p mydb > mydb_backup.sql

# 恢复数据库
mysql -u root -p mydb < mydb_backup.sql
```

### PostgreSQL备份

```bash
# 备份数据库
pg_dump -U postgres mydb > mydb_backup.sql

# 恢复数据库
psql -U postgres mydb < mydb_backup.sql
```

### MongoDB备份

```bash
# 备份数据库
mongodump --db mydb --out ./backup

# 恢复数据库
mongorestore --db mydb ./backup/mydb
```

## 总结

数据库是应用程序的核心组件，选择合适的数据库类型取决于应用场景和需求。关系型数据库适合结构化数据和复杂查询，非关系型数据库适合非结构化数据和高扩展性需求。
