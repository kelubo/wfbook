# 深度学习框架

## 概述

深度学习框架是用于构建、训练和部署深度学习模型的工具，提供了高层API和底层优化，简化了深度学习开发过程。

## PyTorch

### 概述

PyTorch是由Facebook开发的开源深度学习框架，以动态计算图和Pythonic接口著称。

### 安装

```bash
pip install torch torchvision torchaudio
```

### 基础概念

#### 张量（Tensor）

```python
import torch

# 创建张量
x = torch.tensor([1, 2, 3])
x = torch.randn(3, 4)  # 随机张量
x = torch.zeros(3, 4)  # 零张量
x = torch.ones(3, 4)   # 一张量

# 张量运算
y = x + 1
y = x * 2
y = torch.matmul(x, x.T)

# 张量属性
print(x.shape)      # 形状
print(x.dtype)      # 数据类型
print(x.device)     # 设备
```

#### 自动微分

```python
import torch

x = torch.tensor(2.0, requires_grad=True)
y = x ** 2
y.backward()
print(x.grad)  # 输出: 4.0
```

### 神经网络

```python
import torch
import torch.nn as nn

class Net(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super(Net, self).__init__()
        self.fc1 = nn.Linear(input_size, hidden_size)
        self.relu = nn.ReLU()
        self.fc2 = nn.Linear(hidden_size, output_size)
    
    def forward(self, x):
        out = self.fc1(x)
        out = self.relu(out)
        out = self.fc2(out)
        return out

# 创建模型
model = Net(input_size=784, hidden_size=128, output_size=10)

# 损失函数和优化器
criterion = nn.CrossEntropyLoss()
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)

# 训练循环
for epoch in range(10):
    outputs = model(inputs)
    loss = criterion(outputs, labels)
    
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()
```

### 卷积神经网络

```python
import torch.nn.functional as F

class CNN(nn.Module):
    def __init__(self):
        super(CNN, self).__init__()
        self.conv1 = nn.Conv2d(1, 32, kernel_size=3, padding=1)
        self.pool = nn.MaxPool2d(2, 2)
        self.conv2 = nn.Conv2d(32, 64, kernel_size=3, padding=1)
        self.fc1 = nn.Linear(64 * 7 * 7, 128)
        self.fc2 = nn.Linear(128, 10)
    
    def forward(self, x):
        x = self.pool(F.relu(self.conv1(x)))
        x = self.pool(F.relu(self.conv2(x)))
        x = x.view(-1, 64 * 7 * 7)
        x = F.relu(self.fc1(x))
        x = self.fc2(x)
        return x
```

### 保存和加载模型

```python
# 保存模型
torch.save(model.state_dict(), 'model.pth')

# 加载模型
model = Net(input_size=784, hidden_size=128, output_size=10)
model.load_state_dict(torch.load('model.pth'))
model.eval()
```

## TensorFlow

### 概述

TensorFlow是由Google开发的开源深度学习框架，以静态计算图和生产部署能力著称。

### 安装

```bash
pip install tensorflow
```

### 基础概念

#### 张量（Tensor）

```python
import tensorflow as tf

# 创建张量
x = tf.constant([1, 2, 3])
x = tf.random.normal((3, 4))  # 随机张量
x = tf.zeros((3, 4))          # 零张量
x = tf.ones((3, 4))           # 一张量

# 张量运算
y = x + 1
y = x * 2
y = tf.matmul(x, tf.transpose(x))

# 张量属性
print(x.shape)      # 形状
print(x.dtype)      # 数据类型
```

#### 自动微分

```python
import tensorflow as tf

x = tf.Variable(2.0)
with tf.GradientTape() as tape:
    y = x ** 2
grad = tape.gradient(y, x)
print(grad)  # 输出: tf.Tensor(4.0, shape=(), dtype=float32)
```

## PyTorch vs TensorFlow

| 特性 | PyTorch | TensorFlow |
|------|---------|------------|
| 计算图 | 动态 | 静态/动态 |
| 接口风格 | Pythonic | Keras API |
| 调试 | 容易 | 较难 |
| 研究 | 首选 | 次选 |
| 生产部署 | 较难 | 容易 |
| 移动端 | PyTorch Mobile | TensorFlow Lite |

## 模型转换

### PyTorch转ONNX

```python
import torch
import torchvision.models as models

model = models.resnet18(pretrained=True)
dummy_input = torch.randn(1, 3, 224, 224)

torch.onnx.export(model, dummy_input, "resnet18.onnx")
```

### TensorFlow转ONNX

```python
import tensorflow as tf
from tf2onnx import convert

model = tf.keras.applications.ResNet50()
output_path = "resnet50.onnx"

model_proto, _ = convert.from_keras(model, output_path=output_path)
```

## 总结

PyTorch和TensorFlow是目前最流行的深度学习框架。PyTorch适合研究和快速原型开发，TensorFlow适合生产部署和大规模训练。选择哪个框架取决于具体需求和场景。
