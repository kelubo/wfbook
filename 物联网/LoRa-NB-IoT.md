# LoRa与NB-IoT详解

## 概述

LoRa和NB-IoT是两种主流的低功耗广域网络（LPWAN）技术，专为物联网设备远程通信设计，具有低功耗、远距离、大容量的特点。

## LPWAN技术对比

| 特性 | LoRa | NB-IoT |
|------|------|--------|
| 调制方式 | LoRa扩频 | OFDMA |
| 网络架构 | 私有网络 | 运营商网络 |
| 传输距离 | 5-15km（城区） | 10km+ |
| 数据速率 | 0.3-50kbps | 160kbps |
| 功耗 | 极低 | 低 |
| 部署成本 | 较低 | 较高 |
| 覆盖范围 | 需自建网关 | 运营商覆盖 |

## LoRa技术

### LoRa概述

LoRa（Long Range）是一种基于扩频技术的低功耗无线通信技术，由Semtech公司开发。

### LoRa工作原理

#### 扩频技术

- **直接序列扩频**：将数据信号扩展到更宽的频带
- **增益**：扩频增益 = 扩频带宽 / 数据速率
- **优势**：抗干扰能力强，传输距离远

#### LoRa调制参数

| 参数 | 说明 | 范围 |
|------|------|------|
| SF（扩频因子） | 扩频因子，影响速率和距离 | 7-12 |
| BW（带宽） | 信号带宽 | 125/250/500kHz |
| CR（编码率） | 前向纠错编码率 | 4/5, 4/6, 4/7, 4/8 |

### LoRaWAN协议

#### 设备类别

- **Class A**：最低功耗，仅在发送后短暂接收
- **Class B**：定时接收，功耗适中
- **Class C**：持续接收，功耗最高

#### 数据传输流程

1. 传感器发送数据到网关
2. 网关转发数据到网络服务器
3. 网络服务器处理并转发到应用服务器

### LoRa设备

#### 芯片

- **SX126x**：新一代LoRa芯片，支持低功耗
- **SX127x**：经典LoRa芯片，广泛使用

#### 模块

- **ESP32 + LoRa模块**：WiFi+LoRa双模
- **LoRa节点**：专用LoRa传感器节点

### LoRa应用场景

- **环境监测**：空气质量、水质监测
- **智慧农业**：土壤湿度、温度监测
- **物流追踪**：货物位置跟踪
- **智能抄表**：水表、电表远程抄表

## NB-IoT技术

### NB-IoT概述

NB-IoT（Narrow Band IoT）是一种基于蜂窝网络的低功耗广域网络技术，由3GPP标准化。

### NB-IoT工作原理

#### 窄带通信

- 带宽仅200kHz
- 基于LTE技术演进
- 复用现有蜂窝网络基础设施

#### 三种部署模式

- **独立部署（Standalone）**：使用专门的频谱
- **保护带部署（Guard band）**：使用LTE保护带
- **带内部署（In-band）**：使用LTE载波内的资源块


### NB-IoT协议栈

#### 物理层

- OFDMA调制
- 200kHz带宽
- 支持半双工通信

#### 数据链路层

- MAC层：处理接入控制
- RLC层：可靠传输
- PDCP层：加密和压缩

#### 网络层

- IP协议：支持IPv4/IPv6
- NAS层：非接入层信令

### NB-IoT设备

#### 模组

- **移远BC95**：低成本NB-IoT模组
- **华为Boudica**：高性能NB-IoT芯片
- **展锐春藤**：国产NB-IoT芯片

#### 开发板

- **Arduino + NB-IoT扩展板**
- **STM32 + NB-IoT开发板**

### NB-IoT应用场景

- **智能抄表**：水表、电表、燃气表
- **智慧城市**：路灯控制、垃圾桶监测
- **农业物联网**：气象站、土壤监测
- **资产追踪**：车辆、设备定位

## LoRa与NB-IoT对比

### 成本对比

| 项目 | LoRa | NB-IoT |
|------|------|--------|
| 设备成本 | 低 | 中 |
| 网关/基站 | 自建成本低 | 运营商部署 |
| 通信费用 | 无 | 按流量收费 |

### 覆盖对比

| 场景 | LoRa | NB-IoT |
|------|------|--------|
| 城区覆盖 | 需自建网关 | 运营商覆盖 |
| 郊区覆盖 | 网关覆盖范围大 | 依赖基站 |
| 室内覆盖 | 较差 | 较好 |

### 功耗对比

| 模式 | LoRa | NB-IoT |
|------|------|--------|
| 待机电流 | <1μA | <5μA |
| 工作电流 | 10-50mA | 50-200mA |
| 电池寿命 | 5-10年 | 3-5年 |

## 选择建议

### 选择LoRa的情况

- 需要私有网络
- 部署区域没有运营商覆盖
- 低成本、低功耗需求
- 数据量小，传输频率低

### 选择NB-IoT的情况

- 需要广域覆盖
- 依赖运营商网络
- 需要稳定可靠的连接
- 数据量较大，需要确认传输

### 优势

- 覆盖互补：LoRa覆盖盲区由NB-IoT补充
- 灵活选择：根据场景选择合适的技术
- 统一管理：统一平台管理所有设备

## 总结

LoRa和NB-IoT各有优势，选择哪种技术取决于具体应用场景。LoRa适合私有网络、低成本部署，NB-IoT适合广域覆盖、运营商网络。在实际应用中，可以根据需求选择单一技术或混合架构。
