# 量子计算

## 概述

量子计算是利用量子力学现象进行计算的新型计算模式，具有超越经典计算机的计算能力。

---

## 一、量子计算基础

### 1.1 量子比特

```
量子比特（Qubit）
        │
        ├─ 经典比特：0或1
        ├─ 量子比特：0、1或叠加态
        ├─ 叠加态：同时处于0和1的状态
        └─ 量子态：|0⟩、|1⟩或α|0⟩+β|1⟩
```

### 1.2 量子力学原理

```
量子力学原理
        │
        ├─ 叠加原理：量子系统可以处于多个状态的叠加
        ├─ 量子纠缠：两个或多个量子比特之间存在关联
        ├─ 量子隧穿：量子粒子可以穿越能量障碍
        └─ 量子测量：测量会使量子态坍缩
```

### 1.3 量子门

```
量子门
        │
        ├─ 单量子比特门
        │   ├─ Pauli-X门：比特翻转
        │   ├─ Hadamard门：创建叠加态
        │   ├─ Pauli-Z门：相位翻转
        │   └─ Rotation门：旋转量子态
        │
        ├─ 双量子比特门
        │   ├─ CNOT门：受控非门
        │   ├─ SWAP门：交换两个量子比特
        │   └─ Toffoli门：三量子比特门
        │
        └─ 量子电路：量子门的组合
```

### 2.2 量子算法

```
量子算法
        │
        ├─ Shor算法：大数分解
        │   ├─ 经典复杂度：指数级
        │   ├─ 量子复杂度：多项式级
        │   └─ 应用：破解RSA加密
        │
        ├─ Grover算法：无序搜索
        │   ├─ 经典复杂度：O(N)
        │   ├─ 量子复杂度：O(√N)
        │   └─ 应用：数据库搜索
        │
        ├─ 量子傅里叶变换：信号处理
        │   └─ 应用：量子算法基础
        │
        └─ Variational量子算法：混合量子经典算法
            ├─ VQE：变分量子本征求解器
            └─ QAOA：量子近似优化算法
```

```
        ├─ Q#：Microsoft量子编程语言
        └─ PyQuil：Rigetti量子计算框架
```

### 5.2 Qiskit编程示例

```
Qiskit编程示例
from qiskit import QuantumCircuit, execute, Aer

# 创建量子电路
qc = QuantumCircuit(2, 2)

# 添加量子门
qc.h(0)
qc.cx(0, 1)
qc.measure([0, 1], [0, 1])

# 在模拟器上运行
simulator = Aer.get_backend('qasm_simulator')
result = execute(qc, simulator, shots=1024).result()
counts = result.get_counts(qc)

print(counts)
```
