# 网络安全设备

## 概述

网络安全设备是保护网络安全的硬件或软件设备，包括防火墙、入侵检测系统、VPN设备等。

## 防火墙

### 防火墙类型

#### 包过滤防火墙

```
基于IP地址、端口、协议进行过滤
```

```bash
# Linux iptables包过滤
sudo iptables -A INPUT -s 192.168.1.0/24 -p tcp --dport 80 -j ACCEPT
sudo iptables -A INPUT -p tcp --dport 22 -j ACCEPT
sudo iptables -A INPUT -j DROP
```

#### 状态检测防火墙

```
跟踪连接状态，根据状态进行过滤
```

```bash
# Linux iptables状态检测
sudo iptables -A INPUT -m conntrack --ctstate ESTABLISHED,RELATED -j ACCEPT
sudo iptables -A INPUT -m conntrack --ctstate NEW -p tcp --dport 80 -j ACCEPT
```

#### 应用层防火墙

```
分析应用层协议，进行深度包检测
```

### 防火墙部署

#### 边界防火墙

```
部署在网络边界，保护内部网络
```

```
Internet ──► 防火墙 ──► 内部网络
```

#### DMZ区域

```
隔离公共服务，保护内部网络
```

```
Internet ──► 防火墙 ──► DMZ（Web服务器、邮件服务器）
                      └─► 内部网络
```

#### 内部防火墙

```
隔离内部网络区域，限制横向移动
```

### 下一代防火墙（NGFW）

```
集成防火墙、入侵检测、应用控制、VPN等功能
```

## 入侵检测系统（IDS）

### IDS类型

#### 网络入侵检测系统（NIDS）

```
监听网络流量，检测入侵行为
```

```bash
# Snort配置
sudo apt-get install snort

# /etc/snort/snort.conf
var HOME_NET 192.168.1.0/24
var EXTERNAL_NET any

# 启动Snort
sudo snort -A console -i eth0 -c /etc/snort/snort.conf
```

#### 主机入侵检测系统（HIDS）

```
监控主机系统，检测异常行为
```

```bash
# OSSEC配置
sudo apt-get install ossec-hids

# /var/ossec/etc/ossec.conf
<syscheck>
    <directories check_all="yes">/etc,/usr/bin,/usr/sbin</directories>
</syscheck>

# 启动OSSEC
sudo /var/ossec/bin/ossec-control start
```

### IDS部署

#### 端口镜像

```
复制流量到IDS进行分析
```

```
交换机端口镜像配置
interface GigabitEthernet0/1
  port monitor GigabitEthernet0/2
```

#### 串联部署

```
流量必须经过IDS
```

```
Internet ──► IDS ──► 防火墙 ──► 内部网络
```

## 入侵防御系统（IPS）

### IPS特点

```
实时阻断入侵行为，主动防御
```

### IPS部署

```
串联在网络中，实时处理流量
```

```
Internet ──► IPS ──► 内部网络
```

```

```

## VPN设备

### VPN类型

#### IPsec VPN

```
基于IPsec协议的安全隧道
```

```bash
# StrongSwan配置
sudo apt-get install strongswan

# /etc/strongswan/ipsec.conf
conn site-to-site
  left=192.168.1.1
  right=192.168.2.1
  leftsubnet=192.168.1.0/24
  rightsubnet=192.168.2.0/24
  auto=start
```

#### SSL VPN

```
基于SSL/TLS的远程访问
```

```bash
# OpenVPN配置
sudo apt-get install openvpn

# 服务端配置
port 1194
proto udp
dev tun
ca ca.crt
cert server.crt
key server.key
dh dh.pem
server 10.8.0.0 255.255.255.0

# 启动OpenVPN
sudo systemctl start openvpn@server
```

#### WireGuard VPN

```
现代高性能VPN协议
```

```bash
# WireGuard配置
sudo apt-get install wireguard

# /etc/wireguard/wg0.conf
[Interface]
PrivateKey = <服务器私钥>
Address = 10.0.0.1/24
ListenPort = 51820

[Peer]
PublicKey = <客户端公钥>
AllowedIPs = 10.0.0.2/32

# 启动WireGuard
sudo wg-quick up wg0
```

## 负载均衡器

### 负载均衡算法

```
轮询、最小连接数、IP哈希、加权轮询
```

### HAProxy配置

```bash
# HAProxy配置
sudo apt-get install haproxy

# /etc/haproxy/haproxy.cfg
frontend http_front
    bind *:80
    default_backend http_back

backend http_back
    balance roundrobin
    server web1 192.168.1.10:80 check
    server web2 192.168.1.11:80 check
    server web3 192.168.1.12:80 check
```

### Nginx负载均衡

```nginx
http {
    upstream backend {
        server 192.168.1.10:80;
        server 192.168.1.11:80;
        server 192.168.1.12:80;
    }
    
    server {
        listen 80;
        
        location / {
            proxy_pass http://backend;
        }
    }
}
```
